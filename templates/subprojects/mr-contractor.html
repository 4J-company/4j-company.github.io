Module

mr-contractor

A declarative C++ task execution library for modern game development

Overview

mr-contractor transforms how you build complex workflows in C++. With its declarative API and type-safe design, it makes managing task dependencies and parallel execution intuitive and efficient.

Core Features

  • Declarative API for workflow definition
  • Type-safe task dependencies
  • Automatic dependency resolution
  • Parallel task execution

Advanced Features

  • Nested workflow support
  • Conditional task execution
  • Task cancellation and cleanup
  • Progress monitoring and reporting

Example Usage

// Define a simple sequential pipeline
auto pipeline = make_pipeline(
    load_assets(),
    process_textures(),
    generate_materials(),
    build_scene()
);

// Execute the pipeline
pipeline.execute();

Pro Tip

Use the declarative API to define complex workflows in a clear, maintainable way. The library automatically handles dependency resolution and parallel execution.
