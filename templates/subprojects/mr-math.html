Module

mr-math

A lightweight and high-performance linear algebra library for 3D graphics

Overview

mr-math is a modern C++23 linear algebra library specifically designed for 3D graphics and game development. It provides a comprehensive set of mathematical operations with a focus on performance and ease of use.

Core Features

  • Cross-platform support
  • Modern C++23 compatibility
  • Type-safe operations
  • SIMD optimizations

Mathematical Operations

  • Vector and matrix operations
  • Quaternion support
  • Geometric primitives
  • Interpolation functions

Performance

// Example of optimized vector operations
vec3 position = {1.0f, 2.0f, 3.0f};
vec3 velocity = {0.1f, 0.2f, 0.3f};
float deltaTime = 0.016f;

// SIMD-optimized update
position += velocity * deltaTime;

Pro Tip

mr-math's SIMD optimizations provide significant performance improvements for vector operations, making it ideal for real-time graphics applications.
