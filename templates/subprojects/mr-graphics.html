Module

mr-graphics

A high-performance Vulkan rendering library for modern 3D graphics

View on GitHub

Overview

mr-graphics is the core rendering engine of the model-renderer suite, providing a modern, high-performance graphics API built on Vulkan. It handles everything from low-level GPU resource management to high-level scene rendering, enabling developers to create visually stunning applications with optimal performance.

Core Features

  • Full Vulkan 1.3 integration
  • Multi-threaded rendering architecture
  • Optimized render graph system
  • Advanced GPU memory management

Rendering Capabilities

  • Physically-based rendering (PBR)
  • Real-time global illumination
  • Volumetric lighting and fog
  • Advanced post-processing effects

Technical Specifications

Feature Specification
Graphics API Vulkan 1.3
Shader Language GLSL, HLSL (with internal converter)
Memory Management Custom pool allocator with defragmentation
Ray Tracing VK_KHR_ray_tracing_pipeline
Mesh Shading VK_EXT_mesh_shader

Example Usage

// Initialize the graphics context
mr::graphics::Context context;
context.initialize(window.getNativeHandle());

// Create a renderer
mr::graphics::Renderer renderer(context);

// Load a mesh
auto mesh = mr::importer::load_mesh("models/character.gltf");

// Create material
auto material = renderer.createMaterial("shaders/pbr.glsl");
material->setTexture("albedo", loadTexture("textures/albedo.png"));
material->setTexture("normal", loadTexture("textures/normal.png"));
material->setTexture("metallic", loadTexture("textures/metallic.png"));
material->setTexture("roughness", loadTexture("textures/roughness.png"));

// Set up scene and camera
auto scene = renderer.createScene();
auto camera = scene->createCamera();
camera->setPosition({0, 1.7f, 5});
camera->lookAt({0, 0, 0});

// Add mesh to scene
auto entity = scene->createEntity();
entity->setMesh(mesh);
entity->setMaterial(material);

// Main render loop
while (window.isOpen()) {
    renderer.beginFrame();
    scene->update(deltaTime);
    renderer.renderScene(scene, camera);
    renderer.endFrame();
    window.swapBuffers();
}

Pro Tip

For optimal performance, group your draw calls by material and use the render graph to minimize pipeline state changes. This can dramatically improve throughput on modern GPUs.
